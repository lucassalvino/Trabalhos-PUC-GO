ao executar o programa, acesse o menu opções -> Carregar Tabela, navegue até onde se encontra o arquivo com as produções e a tabela de entrada. Conforme segue arquivo de exemplo ("teste.txt").

Caso seja carregado sem nenhum erro. a tabela será exibida na tela inicial do programa.

Digite a palavra no input line abaixo da tabela, para acompanhar o teste da palavra na tabela, clique em Próximo passo.
A cada click uma nova linha será exibida na tabela de saída. Acompanhe a execução da mesma por essa tabela. Caso a palavra não seja válida será exibido um message box informando o erro ocorrido.

Caso a palavra seja válida, também será exibido um message box infromando que a palavra é válida.]
