Jogo da cobrinha
PUC Goiás

Utilizar QtCreator para realizar a compilação do jogo (função getch() da class UTILIDADES funciona somente para LINUX)
teclas movimentação: W | S | D | A

##########################################################
Arquivo config.txt //necessário
Contem as configurações gerais do jogo
Primiera Linha: dimensão XxY da matriz do jogo contem dois calores inteiros correspondentes a quantidade de  linhas e colunas do mapa
Segunda Linha: Contem o modo gráfico do jogo: 1-> modo texto | 2-> modo gráfico


##########################################################
Arquivo jogador.txt //caso não exista será solicitado os dados ao inicializar
Contem os dados do jogador
Primeira Linha: Nome do jogador
Segunda Linha: Inteiro contendo a pontuação do jogador
Terceira Linha: Numero de mortes do jogador
Quarta Linha: Numero inteiro com o maior recorde do jogador
