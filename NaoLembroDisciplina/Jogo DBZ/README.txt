1º cola na pasta pessoal
2º abre o terminal
3º digite "cd" sem aspas e tecle enter
4º digite "make run" sem aspas e tecle enter
5º irá aparecer uma nova janela, não feche a nova janela, volte para o terminal e selecione a opção 1 (Digite "1" e tecle enter")
6º volte para a a janela que abriu e já pode jogar.

jogabilidade:
jogador 1: W A S D X E C
jogador 2: 8 5 6 4 7 9 3 (teclado numerico)


dependencias :
→ allegro = sudo apt-get install liballegro-dev
→ g++     = sudo apt-get install g++
